Política de privacidad de Casual Learn

La aplicación Casual Learn respeta la privacidad de sus usuarios. La identificación se realiza a través de Firebase Authentication por lo que no se almacena directamente información personal de los usuarios. Si los usuarios se identifican mediante Firebase los metadatos de las respuestas que proporcionen se utilizarán para personalizar las tareas propuestas y adecuarlas a sus gustos. Estos datos podrán ser utilizados para investigaciones internas, pero no serán compartidos con terceros con fines comerciales.

Los metadatos de la respuesta del usuario estarán compuestos por la respuesta textual, la posible valoración que haya podido dar el usuario a la tarea, el instante de inicio y finalización y el número de elementos multimedia que haya podido generar. No se envían ni fotografías ni vídeos desde los dispositivos móviles de los usuarios a los servidores de Casual Learn. Tampoco se envía de forma precisa la posición del usuario en ningún momento. El usuario que lo desee podrá recuperar y/o eliminar los datos que haya podido generar enviando un correo electrónico a esta dirección.

Posición y el servicio en segundo plano de Casual Learn

Los usuarios identificados recibirán notificaciones cuando estén cerca de lugares donde existan tareas. Para ello, Casual Learn necesita conocer la ubicación de los usuarios aunque la aplicación no se encuentre en primer plano. No se envía directamente la posición del usuario: a través de Internet se realiza la solicitud de las tareas de una zona en la que se encuentra el usuario. Este proceso únicamente se inicia cuando el usuario salga de la zona que tenga descargada en su móvil o cuando se den por inválidas las tareas almacenadas (cuando pase un día desde que se almacenaron). El envío de esta información llega a los servidores del sistema Casual Learn a través de un protocolo seguro. Para personalizar el orden de prioridad de las tareas se utiliza Recombee. A Recombee se le envía un identificador único del usuario, por lo que no se comparte información personal del usuario con esta herramienta externa.

El tiempo entre notificaciones es configurable por el usuario. Por ello, Casual Learn únicamente obtiene la posición del usuario cuando el temporizador vence. Se recupera la posición de manera periódica hasta que se le notifique al usuario una nueva tarea. No se almacena el recorrido que realiza el usuario. Además, si el usuario que esté identificado desea detener esta característica podrá activar el modo «No molestar» en los ajustes de la aplicación. Con el modo «No molestar» activo, la aplicación no volverá a recuperar la ubicación del usuario en segundo plano hasta que el usuario lo desactive.

Porfolio del sistema

Con las respuestas que proporcione el usuario el sistema Casual Learn genera un porfolio accesible a través de Internet. La activación de este porfolio se realiza de manera proactiva desde los ajustes de la aplicación Casual Learn. El usuario puede eliminar la publicación de este porfolio en cualquier momento desde la aplicación. También podrá elegir de manera individual qué respuesta mostrar o no mostrar de manera individual.

En nuestro compromiso por mantener la privacidad de los usuarios, y debido a que no se almacena información con la que poder identificar a los usuarios del sistema, en el porfolio no aparece ninguna información sobre la autoría del porfolio salvo la que pueda indicar el propio usuario en una respuesta textual. Casual Learn no se responsabiliza del contenido que pudieran dar los usuarios en las respuestas.
